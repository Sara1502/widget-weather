# 🌤️ Weather App

App web de previsão do tempo utilizando a API OpenWeather.

## 💻 Funcionalidades

- Solicitação de localização do usuario;
- Utlização de API de clima/tempo OpenWeather para obter a temperatura e clima real da localização do usuario;
- Trocar de cor tema e background conforme a temperatura e o clima:
-  Abaixo de 20°C tema frio com background de neve;
-  Entre 20°C e 26°C tema ameno com backgroud com flores;
-  Acima de 26°C tema quente com background ensolaraso;
- Troca de backgroud em relação ao clima:
-  Quando esta chovendo, background com chuva;
-  Quando não esta chovendo, background com sol;
